# Sonata · 250-Lesson Curriculum

All 250 lessons. Title + one-sentence goal.
Use this with `LESSON_PLANNING_WORKSHEET.md` to plan each one.

**Graduation piece:** Moonlight Sonata 1st movement (full). Henle grade 4 — honestly achievable in 250 structured lessons of daily practice.

Legend:
- 🎯 = Act boss (end of 5-lesson unit)
- 👑 = Tier boss (end of 25/50-lesson tier)
- 🏆 = Graduation (L250)

## How to use this file

1. Pick a lesson number (start with L3).
2. Open `LESSON_PLANNING_WORKSHEET.md`.
3. Copy the title and goal from this file to fill in boxes 1, 2, 3.
4. Fill in the other 5 boxes (vibe, prereqs, piece, metaphor ideas).
5. Paste the filled worksheet into a chat with Claude. Claude writes the 16-page lesson.
6. Read, approve, ship. Move on.

The curriculum is planned. The writing is a process. One lesson at a time.
